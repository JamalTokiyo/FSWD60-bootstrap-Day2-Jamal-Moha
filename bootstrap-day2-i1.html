What is Git?

GitHub Inc. is a web-based hosting service for version control using Git. It is mostly used for computer code. It offers all of the distributed version control and source code management functionality of Git as well as adding its own features

HTML

HTML is the HyperText Markup Language for describing web pages. HyperText is the method by which you move around on the web — by clicking on special text called hyperlinks which bring you to the next page. The fact that it is hyper just means it is not linear — i.e. you can go to any place on the internet whenever you want by clicking on links — there is no set order to do things in. HTML is not a programming language

CSS3

CSS stands for Cascading Style Sheet. CSS is based on a set of rules that describe the priority of how styles are rendered on the page. The rules can be as simple as defining colors, sizes, fonts, alignment etc. The newest version of CSS - CSS3 goes beyond the basic rules offering animation and even a level of interactivity.s

JavaScript

JavaScript is a programming language used to make web pages interactive. In conjunction with HTML5 and CSS3 you will be able to create rich, interactive and dynamic GUI (Graphic User Interface).

JQUERY

jQuery is a library that makes writing JavaScript code a lot easier, with a nice motto: “Write less, do more”. It helps you build cross-browser websites or applications.

Bootstrap

Bootstrap is a free open source front-end framework - Mark Otto and Jacob Thornton are creators of Bootstrap. The term framework refers to a foundation of pre-built tools, software, code, and other resources we can use to create a project more easily than if we created it from scratch. When Bootstrap was initially released, Mark and Jacob both were employees at Twitter
